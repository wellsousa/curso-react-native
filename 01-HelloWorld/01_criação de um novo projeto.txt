PARA INSTALAR O REACT-NATIVE GLOBALMENTE:

sudo npm i -g react-native-cli

PARA CRIAR UM NOVO PROJETO REACT-NATIVE VA DENTRO DO DIRETORIO
ONDE O PROJETO SERÁ CRIADO E DIGITE:

sudo react-native init [NomeProjeto]

OBS: O NOME DO PROJETO NÃO ACEITA CARACTERES ESPECIAS E NEM '-' E '_'.
UM NOME VÁLIDO É POR EXEMPLO 'MeuPrimeiroProjeto'
