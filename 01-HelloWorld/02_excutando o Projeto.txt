PARA EXECUTAR O PROJETO VOCÊ DEVE SEGUIR OS PASSOS:

- EXECUTE O EMULADOR
- DE START NO PROJETO
- CHAME O METODO RUN-ANDROID


EXECUTANDO O EMULADOR

PARA VISUALIZAR OS EMULADORES DISPONIVEIS:

emulator -avd -list-avds

PARA INICIAR O EMULADOR:

emulator -avd [nome-do-emulador]
emulator -avd Nexus_5X_API_29_x86

START NO PROJETO:

VÁ DENTRO DO DIRETORIO DO PROJETO E EXECUTE O COMANDO:

react-native start

RODE O ANDROID:

TAMBEM DENTRO DO DIRETORIO DO PROJETO EXECUTE:

react-native run-android
